Introduction¶

Intonatang is the analysis code for Claire Tang’s speech intonation project.

Speech intonation plays an important role in vocal communication. It can be used to emphasize certain words in a sentence or to change a statement into a question. This project explores how intonational pitch contours are represented in human auditory cortex.
